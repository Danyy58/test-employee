# Отчёт
Для оптимизации запроса в задании 6 были проделаны следующие действия:
1. Создание нового столбца с инициалами;
2. Создание индексов по столбцам со значением пола и с инициалом.

Создание столбца с инициалом и индексов по полу и этому инициалу позволило базе данных выполнять поиск по заранее вычисленным и отсортированным значениям, исключив полное сканирование таблицы и тем самым значительно ускорив выполнение запроса.
